## Installing packages

```bash
make install
```

## 1st terminal: MySQL

```bash
cd src
make run-mysql
```

## 2nd terminal: Backend

```bash
cd src
make run-backend
```

## 3rd terminal: Frontend

```bash
cd src
make run-frontend
```
